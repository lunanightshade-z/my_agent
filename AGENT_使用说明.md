## 🚀 快速开始

### 1. 访问应用

打开浏览器访问：
```
http://localhost:28888/agent
```

或者通过外网IP访问：
```
http://8.130.129.37:28888/agent
```

### 2. 开始对话

在Agent页面的输入框中输入问题，例如：
- "帮我获取10条最新新闻"
- "找一些关于AI的新闻"  
- "搜索包含'科技'关键词的新闻"

### 3. 观察工具调用

智能体会自动判断是否需要调用工具，你会看到：
```
🔧 调用工具: fetch_rss_news
   参数: {"max_articles": 10}
   
✅ 工具 fetch_rss_news 执行完成
```

## 🐛 常见问题

### Q: 看到502错误怎么办？
A: 等待几秒后刷新页面，后端容器可能正在启动。

### Q: 工具调用失败？
A: 部分RSS源可能无法访问，这是正常的，智能体会使用可用的源。

### Q: 响应很慢？
A: RSS获取需要时间（约30秒），这是正常的。
