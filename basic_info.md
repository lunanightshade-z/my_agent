
# 项目介绍
这是一个智能体demo，涵盖简单聊天（chat页面）和AI日报智能体（agent页面）

## 页面导航说明
- 首页（Home）：展示项目介绍和特性，"Awaken The Magic" 按钮点击后跳转到 Agent 页面
- Chat 页面：简单聊天功能
- Agent 页面：AI日报智能体功能
