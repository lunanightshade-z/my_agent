# 智能体文档工具增强（2026-01-23）

## 变更概述
- 新增PDF文本提取、CSV概览分析、行动项提取三类工具，并在智能体服务中注册
- 增加文档上传与上传记录查询接口，支持返回`file_id`供工具调用
- 新增文件存储与元数据管理模块，统一上传目录与校验规则
- 更新系统提示词，补充文档工具使用原则
- 补充对应单元测试用例

## 接口与工具使用
1. 上传文档：`POST /api/agent/uploads`（multipart form-data）
2. 查询上传记录：`GET /api/agent/uploads`
3. 对话中使用`file_id`触发工具：
   - `extract_pdf_text`：提取PDF文本并生成摘要/要点
   - `analyze_csv_file`：输出CSV结构、样例行、数值列统计
   - `extract_action_items`：从会议纪要文本中提取行动项
