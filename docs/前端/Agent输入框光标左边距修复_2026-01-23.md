# Agent 输入框光标左边距修复

## 问题描述
Agent 页面的输入区域中，闪烁的光标距离左侧边缘太近，影响用户体验。

## 修复方案
在 `Agent.jsx` 的 textarea 组件中添加了左右内边距 `px-4`（左右各 16px），使光标与边缘保持适当距离。

## 修改内容
- **文件**: `frontend/src/pages/Agent.jsx`
- **位置**: 第 580-588 行（textarea className）
- **修改**: 在 className 中添加 `px-4`，与现有的 `py-3` 配合使用

## 修改前
```jsx
className="
  flex-1 bg-transparent border-none outline-none resize-none 
  py-3 max-h-32 text-slate-700 placeholder:text-slate-400/70 
  ...
"
```

## 修改后
```jsx
className="
  flex-1 bg-transparent border-none outline-none resize-none 
  py-3 px-4 max-h-32 text-slate-700 placeholder:text-slate-400/70 
  ...
"
```

## 效果
- 光标现在距离左侧边缘有 16px 的间距，视觉上更加舒适
- 文本输入时不会紧贴边缘，提升用户体验
- 保持了原有的样式风格和响应式布局
