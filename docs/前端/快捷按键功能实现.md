# 快捷按键功能实现

## 概述
为 Chat 和 Agent 两个页面添加了快捷按键功能，用户点击预设的问题即可快速发送消息。

## 最新更新（2026-01-27）
- ✅ 快捷按键位置调整：从页面中间移动到输入框上方
- ✅ 模型选择器位置调整：从右上角移动到输入框左侧
- ✅ 优化了布局和对齐方式

## 实现内容

### 1. 创建快捷按键组件
- **文件**: `frontend/src/components/QuickActions.jsx`
- **功能**: 通用的快捷按键组件，支持不同主题样式
- **特性**:
  - 支持 Chat 和 Agent 两种主题样式
  - 动画效果（framer-motion）
  - 禁用状态处理
  - 响应式布局，支持长文本换行

### 2. Chat 页面集成
- **文件**: `frontend/src/components/layout/AppLayout.jsx`
- **快捷按键列表**:
  1. 什么是LLM Agent？
  2. Agent记忆系统如何搭建？
  3. 大模型+AR有哪些有意思的场景应用？
- **显示位置**: 当消息列表为空时，在输入框上方显示
- **模型选择器位置**: 输入框左侧
- **样式**: 使用 Chat 主题的深色风格（amber 色系）

### 3. Agent 页面集成
- **文件**: `frontend/src/pages/Agent.jsx`
- **快捷按键列表**:
  1. 根据最近的新闻写一篇日报，然后根据内容给出你的独特的有新意的总结。
  2. 总结近期的民生相关的新闻并给出锐评。
- **显示位置**: 当消息列表为空时，在输入框上方显示
- **模型选择器位置**: 输入框左侧（从顶部标题区移动）
- **样式**: 使用 Agent 主题的浅色风格（teal 色系）

### 4. 功能优化
- **Agent 页面**: 修改 `handleSendMessage` 函数，支持直接传入消息文本参数
- **Chat 页面**: `handleSendMessage` 函数已支持消息参数，无需修改
- **文本处理**: 优化长文本显示，支持自动换行

## 技术细节

### 组件接口
```javascript
<QuickActions
  actions={Array<string>}        // 快捷按键文本数组
  onActionClick={Function}       // 点击回调函数，接收 action 文本作为参数
  disabled={boolean}             // 是否禁用（流式响应时禁用）
  theme="chat" | "agent"         // 主题样式
/>
```

### 样式特点
- **Chat 主题**: 深色背景，amber 色系，玻璃态效果
- **Agent 主题**: 浅色背景，teal 色系，半透明效果
- **动画**: 渐入动画，悬停缩放效果

2. **快速开始**: 提供常用问题的快捷入口
