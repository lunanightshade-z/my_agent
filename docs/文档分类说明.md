# docs 文档分类说明

> 更新日期：2026-01-27  
> 本文档说明 `docs/` 下的中文分类结构，便于按主题查找文档。

## 分类结构概览

| 分类文件夹     | 说明                     | 原英文目录   |
|----------------|--------------------------|--------------|
| **后端**       | 智能体、RSS、Chat、多模型等后端技术文档 | backend      |
| **部署**       | Docker、docker-compose、镜像与发布相关 | deployment   |
| **设计**       | 前端样式指南、设计规范、重构总结       | design       |
| **前端**       | 前端功能、修复、性能、样式与主题文档   | frontend + 前端 |
| **版本控制**   | Git 推送、提交总结、发布记录         | git          |
| **项目说明**   | 项目结构、快速开始、变更记录、优化与交付 | init         |
| **其他**       | 架构、故障排查、迁移、测试等综合文档   | other        |
| **参考**       | 参考代码片段（如 .jsx）             | ref          |

## 前端子分类

- **前端/问题修复**：Bug 修复、错误排查、React/浏览器相关问题
- **前端/性能优化**：性能优化说明、测试指南、输入框延迟等
- **前端/样式与主题**：主题升级、赛博朋克/黑色系样式、补充修复

## 文档命名约定

- 原有中文标题或「中文_日期」形式已保留
- 原英文学名已统一改为简明中文名（如 `agent_api_integration.md` → `智能体API集成文档.md`）
- 日期后缀（如 `_2026-01-27`）未改动，便于与提交、版本对应

## 查找建议

- 查智能体 / Agent 实现与 API → **后端**
- 查部署、Docker、镜像与发布 → **部署**
- 查界面风格、配色、组件视觉 → **设计**、**前端/样式与主题**
- 查前端 Bug、报错、兼容性 → **前端/问题修复**
- 查性能与加载优化 → **前端/性能优化**
- 查项目结构、新手上手、变更记录 → **项目说明**
- 查架构改造、迁移、测试与综合总结 → **其他**
